# Support

animsmith uses GitHub issues for project support. GitHub Discussions are
not enabled for this repository.

## Questions

For usage questions, open a question issue:

https://github.com/mmannerm/animsmith/issues/new

Include the command you ran, the asset format, the animsmith version, and
what you expected to happen.

## Bugs

Use the bug report template:

https://github.com/mmannerm/animsmith/issues/new?template=bug_report.yml

Please include a minimal public or synthetic asset when possible. Do not
attach licensed assets unless you have permission to share them.

## Documentation And Examples

Use the documentation-gap template for missing, stale, or confusing docs:

https://github.com/mmannerm/animsmith/issues/new?template=documentation_gap.yml

Example and tutorial gaps belong there too; the issue should say what
asset type, command, or embedding workflow the example needs to cover.

## Feature Requests

Use the feature request template:

https://github.com/mmannerm/animsmith/issues/new?template=feature_request.yml

Describe the game-animation workflow you want to support and whether it
belongs in the CLI, a Rust crate, the report, or documentation.

## Security

Do not report vulnerabilities in public issues. Use the private reporting
path in [SECURITY.md](SECURITY.md).
